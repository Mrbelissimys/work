## Описание
Python-утилита (CLI) для аудита безопасности серверов на базе Linux. Скрипт парсит системные логи (auth.log) с помощью регулярных выражений, собирает неудачные попытки авторизации по SSH и формирует сводный отчет о самых активных атакующих IP-адресах.
Может использоваться как первый этап для автоматической блокировки (передачи IP в iptables/Fail2Ban).

## Зависимости
* Python 3.6+
* Стандартные библиотеки (`re`, `collections`, `argparse`) — установка дополнительных модулей не требуется.

## Как запустить
Скрипт поддерживает аргументы командной строки. По умолчанию читает `/var/log/auth.log` и выводит топ-5 адресов.

```bash
# Базовый запуск (могут потребоваться права sudo для чтения лога)
sudo python3 ssh_brute_analyzer.py

# Указать свой файл лога и вывести топ-10 адресов
python3 ssh_brute_analyzer.py --file ./my_custom_auth.log --limit 10
