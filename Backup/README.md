

## Описание
Bash-скрипт для создания инкрементальных архивов файлов веб-проекта (или баз данных) с автоматической ротацией старых копий (хранение 7 дней) и отправкой статуса выполнения в Telegram. 
Позволяет моментально узнавать о проблемах с резервным копированием без необходимости вручную проверять логи.

## Зависимости
* Linux (Bash)
* Утилиты: `tar`, `curl`, `find`
* Созданный Telegram-бот (через BotFather) и Chat ID

## Как запустить
1. Скопируйте скрипт на сервер: `/usr/local/bin/backup_with_tg_alert.sh`
2. Впишите ваши `TG_BOT_TOKEN` и `TG_CHAT_ID`.
3. Сделайте файл исполняемым: `chmod +x /usr/local/bin/backup_with_tg_alert.sh`
4. Добавьте в CRON для ежедневного запуска в 03:00 ночи:
```bash
0 3 * * * /usr/local/bin/backup_with_tg_alert.sh >> /var/log/backup.log 2>&1
