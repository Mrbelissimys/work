# AD Bulk User Creator

## Описание
Скрипт на PowerShell для автоматизированного массового заведения учетных записей в Active Directory на основе выгрузки из HR-системы (CSV-файл). 
Решает проблему рутины при онбординге большого количества новых сотрудников. Включает проверку на дубликаты, принудительную смену пароля при первом входе и подробное логирование.

## Зависимости
* Windows Server / Windows 10+ (RSAT)
* Модуль `ActiveDirectory` для PowerShell
* Файл `users.csv` с разделителем `;` (колонки: FirstName, LastName, Department, Title)

## Как запустить
1. Отредактируйте переменную `$TargetOU` в скрипте под структуру вашего домена.
2. Запустите скрипт в консоли PowerShell с правами администратора домена:
```powershell
.\Create-AdUsers.ps1 -CsvPath "C:\path\to\users.csv"
